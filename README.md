# UIU CareerHub - Job Portal UI

A pixel-perfect, modern Job Portal interface designed to connect talent with opportunities. This project focuses on a clean user experience (UX) with a structured layout for job seekers to filter and find relevant positions efficiently.


- Dynamic Job Filtering: Sidebar filters for Job Type (Full-time, Internship, Remote) and Department.
- Categorized Job Cards: Distinct visual styles for different job categories using custom color coding.
- Responsive Navigation: A functional header with links to Jobs, Internships, and Resources.
- Clean Typography: Focused on readability and professional aesthetics.
- Salary & Deadline Indicators: Clear metadata display for quick scanning of job opportunities.

 🎨 Tech Stack

- HTML5: Semantic structure for better accessibility and SEO.
- CSS3: Custom styling using Flexbox for layout management and precise color-coding based on UI specifications.

 🛠️ Color Palette

The design utilizes a specific professional color scheme:
- Primary Blue: `#205BCB` (Buttons & Actions)
- Background: `#F5F6FB` (Soft grey-blue for reduced eye strain)
- Status Tags: - Full-time/Remote: `#E2EBFA`
  - Onsite: `#D6F2E7`
  - Internship: `#FDEBC0`
